AI Salary Estimator

Sistem Prediksi Nilai Tawar Karir Mahasiswa Berbasis Machine Learning

Tentang Proyek AI

Tujuan Aplikasi

Aplikasi web ini dirancang untuk menaksir estimasi gaji awal (Fresh Graduate) lulusan IT/Sains yang beralih ke ranah teknologi. Kecerdasan Buatan (AI) menilai kandidat berdasarkan kompetensi portofolio dan menyeimbangkannya dengan standar ekonomi lokal di Indonesia.

Arsitektur Model (Linear Regression)

Otak dari sistem ini menggunakan algoritma Linear Regression (Regresi Linear). Metode prediksi Machine Learning ini dipilih karena sifatnya yang transparan dalam memetakan hubungan kausalitas (sebab-akibat) antara metrik akademik secara langsung terhadap taksiran gaji numerik.

Proses Training Model

  • Data Mining: Menggunakan dataset profil rekrutmen elite berisi 9.000 sampel lulusan universitas global.
  • Pre-processing: Filter mahasiswa yang diterima kerja dan penerapan One-Hot Encoding pada kolom kategorikal (perusahaan dan jabatan).
  • Feature Selection: Demi mencegah overfitting, AI dirampingkan untuk hanya berfokus pada 5 fitur paling krusial: IPK, Level Kampus, Skor Koding, Tipe Perusahaan, dan Posisi.
  • Evaluasi Metrik: Menggunakan proporsi pemisahan data 80:20 (Train-Test Split), model berhasil mempertahankan tingkat akurasi (R² Score) sebesar 83.13%.
